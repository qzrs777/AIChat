更新内容：
本次更新不仅为用户带来了表现力更丰富的专属微调模型，还面向 Mod 开发者全面开放了 AIChat 的底层交互 API，大幅提升了生态扩展性。

 1. 全新 さとね (Satone) 情感微调模型
基于 Qwen3.5-4B 深度微调，让 AI 的回复搭配生动的肢体语言：

精准的情感捕捉：模型原生支持 19 种精细化情感标签（涵盖开心、思考、害羞、紧张、疲倦等）。

丰富的动作映射：情感标签可自动且随机地映射至 78 个精选动画。内置共计 118 个完整动画库，覆盖待机、工作、休息、剧情及互动等多种场景。

下载途径：已同步更新至百度网盘与本站 Release 附件。

 2. 全面开放外部调用 API (面向开发者)
其他 Mod 现在可以通过标准公开接口（GUID: com.username.chillaimod）与 AIChat 进行深度联动：

核心功能控制：支持外部插件直接读取/写入/落盘配置、控制台显隐切换以及清理 AI 记忆。

多模态对话触发：支持通过 API 随时发起文本对话或语音对话（直接传入 WAV 字节）。

结构化数据回调：新增 ConversationCompleted 事件，对话结束后自动返回情绪标签、语音/字幕文本、TTS 状态及错误码等详尽数据，方便外部逻辑接管。

最佳实践推荐：强烈建议开发者使用 BepInEx Chainloader 进行“软依赖”调用（未安装时不抛异常）。参考代码可查看本仓库的 ChillPatcher 桥接实现。
